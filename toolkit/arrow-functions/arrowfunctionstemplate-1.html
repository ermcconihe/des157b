Change the Picture

I borrowed some of my code from the every picture project in the previous interactive media class. This basically makes a photo fade out on hover.

Named Function

            function changePicture(change){
                const thisPicture = change.target;
                thisPicture.className = 'top fadeout';
            }
            eachTop.addEventListener('mouseover', changePicture);

Function Expression

            const changePicture = function(change){
                const thisPicture = change.target;
                thisPicture.className = 'top fadeout';
            }
            eachTop.addEventListener('mouseover', changePicture);

Arrow Function

            const changePicture = (change) => {
                const thisPicture = change.target;
                thisPicture.className = 'top fadeout'; 
            }
            eachTop.addEventListener('mouseover', changePicture);
